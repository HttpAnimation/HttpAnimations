Info:

For those new to emulation these games or files run on emulators such as Demul, you drag and drop the bios to the respected file and then run a rom and play. This is not a complete collection just a selection of the most interesting ones. This includes US UK and Japan releases. Some are not English translated. There are a number of titles not in this collection those mostly, are duplicates on other systems. Bios files are included as well as the Utopia CD Loader V1.1 Includes hombrews: Capcom Fighting Collection has about 14 games on it Capcom vs SNK Millennium Collection contains 4 games Halo Revamped is a port of the PSP quake mod Halo Revamped for dreamcast made in 2011 it works best on actual consoles and is known to cause tons of crashes on emulators. Inside are 2 copies of Hellgate a rarer UK horror game. The accel99 version is more polished but has a slower framerate. The IND version is the one you see around the web. Mostly CDI formats some GDI. If you are having problems getting Demul to launch, add all the BIOS files except Utopia to "Dc.zip" without the " place this in your ROMs folder in demul and then launch the program. If Utopia is in the bios zip it won't launch
